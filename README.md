# Lente

**Pesquisa qualitativa em vídeo com IA.** Entrevistas que se aprofundam sozinhas.

Você define o roteiro e o contexto. A Lente conduz cada entrevista com perguntas de
follow-up adaptativas, transcreve, e devolve uma síntese com recortes em vídeo das citações
que sustentam cada insight.

**Ao vivo:** [lente.pereirasaraiva.com](https://lente.pereirasaraiva.com) · demo sem cadastro.

## O problema
A entrevista qualitativa vira transcrição crua de 15+ minutos por pessoa. O insight fica
enterrado, e a evidência (o momento exato em que o usuário travou) se perde no texto.

## Como funciona
1. **Defina o estudo:** briefing de negócio, contexto e roteiro. A IA usa isso para guiar
   cada conversa.
2. **Capture em vídeo:** link único por entrevistado; pergunta a pergunta, com follow-up
   quando a resposta merece.
3. **Síntese com evidência:** temas recorrentes, clipes citáveis e recomendações ancoradas
   nos vídeos (cada insight abre o clipe original).

## Stack
React · TanStack · Tailwind · Supabase · Cloudflare. IA para condução, transcrição e síntese.
